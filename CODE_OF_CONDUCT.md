# Code Of Conduct

## Our Standard

We want Tinkerloop to be a professional, technically rigorous project.
Participants are expected to:

- be respectful and direct
- debate ideas without personal attacks
- assume good intent while still insisting on technical clarity
- avoid harassment, discrimination, or intimidation

## Unacceptable Behavior

Examples include:

- personal insults or derogatory language
- harassment or stalking
- publishing private information without consent
- sustained disruption of discussions or reviews

## Enforcement

Project maintainers may remove, edit, or reject contributions and discussion
that violate this code of conduct.

## Reporting

While the project remains private, report concerns through the maintainers'
existing internal channels.
If and when the project becomes public, this file should be updated with a
public contact path.
